개인적으로 VueJS 기능을 테스트 및 사용방법을 예제로 구현합니다.

### Install
```
$bower install bootstrap vue moment moment-timezone rivets
```


### bower.json 파일의 의미
```
name: (필수값) 팩키지명
version: 버전 정보
main: 패키지의 엔드포인트
ignore: 패키지를 설치할 때 bower가 무시할 파일 목록
dependencies : 프로덕션에서 사용할 패키지 의존성 정보
devDependencies : 개발용 패키지 의존성 정보

```
